Security Policy

Responsible disclosure

If you discover a security issue on szabee.me, please report it so I can fix it quickly.

How to report

Send details to me@szabee.me or reach out on Discord.

What to include

  • Steps to reproduce the issue.
  • Proof-of-concept or screenshots if applicable.
  • Any relevant logs, payloads, or URLs.

Scope

Please avoid testing on third-party services and do not access data that is not yours. Use good-faith testing and keep any findings confidential until a fix is available.

Thanks for helping keep szabee.me safe.
